Github Profile
ReLive

ReLive is an Android application that utilizes the Facebook API to generate a quiz where the user is given a random past status or profile picture of one of their friends, and is offered 4 names of friends to guess the source. Unfortunately, due to restrictions with the Facebook API, the app can only retrieve data from friends that have also given permission to the app, but past that, the app works flawlessly.

Analysis of Donations from Pharmaceutical Companies to Hospitals/Doctors

This application uses R to display calculations from a data set of payments and gifts given from pharmaceutical companies to both hospitals and doctors. The output is displayed on a website that is hosted by a service called Shiny. The website itself was not coded directly, but rather was generated by tools built to work with R called Knitr and Shiny.
